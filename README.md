Este projeto tem como propósito desenvolver um modelo de classificação automática de sentimentos  em  textos,  aplicando  conceitos  fundamentais  de  Python,  Machine  Learning  e MLOps. A iniciativa parte de um problema de negócio: uma empresa de e-commerce que deseja automatizar  a  análise  de  feedbacks  de  clientes  para  reduzir  custos,  aumentar  a  eficiência  e acelerar a tomada de decisão.O trabalho é estruturado de forma modular, cobrindo todas as etapas do ciclo de vida de um  projeto  de  Ciência  de  Dados.  Após  a  definição  do  problema  e  objetivos,  o  script  realiza  a importação   das   bibliotecas   essenciais   (Pandas,   Scikit-learn,   Seaborn,   entreoutras)   e   o carregamento dos dados a partir de um arquivo CSV contendo reviews de produtos. Em seguida, conduz   uma   análise   exploratória   detalhada   (EDA)   para   compreender   a   distribuição   dos sentimentos e verificar a presença de valores ausentes.Na  etapa  de  limpeza  de  dados,  é  implementada  uma  função  de  normalização  de  texto (remoção de acentos, pontuações e caracteres especiais), preparando o conteúdo textual para a engenharia de atributos, onde as classes “positivo” e “negativo” são mapeadas para  rótulos numéricos  (1  e  0).  O  conjunto  de  dados  é  então  dividido  em  amostras  de  treino  e  teste  para garantir uma avaliação correta da performance do modelo.
O núcleo do projeto é o pipeline de modelagem preditiva, que integra três componentes principais em um único objeto Python:-TF-IDF Vectorizer para transformar o texto em representações numéricas;-StandardScaler para padronização dos vetores esparsos;-Logistic Regression como algoritmo de classificação supervisionada.O modelo é otimizado com GridSearchCV, explorando combinações de hiperparâmetros para maximizar a acurácia. Após o treinamento, o desempenho é avaliado por meio de métricas como Acurácia, Precisão, Recall e F1-Score, além da visualização da matriz de confusão. O modelo final é então salvo em disco com a biblioteca joblib, permitindo reuso e integração com sistemas externos.Por fim, o projeto inclui  uma simulação de  deploy (MLOps), utilizando uma função que carrega o modelo treinado e classifica novos reviews de forma automática. Essa fase demonstra como o modelo poderia ser integrado em pipelines reais de produção, entregando valor imediato para as áreas de marketing, suporte e gestão da experiência do cliente.Trata-se  de  um  projeto  completo  de  Data  Science  para  análise  de  sentimentos,  que combina teoria e prática em Python, abordando desde o tratamento de dados até a automação de  predições,  com  uma  estrutura  profissional  alinhada  às  boas  práticas  de  Data  Science  e Machine Learning.
